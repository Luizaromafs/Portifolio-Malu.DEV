# Portifolio-Malu.dev
Este é o meu portfólio pessoal, desenvolvido para apresentar meus projetos, minhas habilidades como desenvolvedora
front-end júnior e minha evolução no mundo da tecnologia. O objetivo é reunir tudo em um espaço organizado, moderno
e responsivo, permitindo que recrutadores e empresas conheçam melhor meu trabalho.

# Sobre o Projeto:

O portfólio foi criado com foco em:
Responsividade e boa experiência do usuário;
Design limpo, organizado e fácil de navegar;
Código simples, porém bem estruturado;
Demonstração prática das minhas habilidades com front-end;
Além de apresentar meus projetos, o portfólio também conta com uma seção sobre mim, meus contatos profissionais e
links importantes.

# Tecnologias Utilizadas:

HTML5 – Estrutura;
CSS3 – Estilização e responsividade;
JavaScript – Interações e lógica da página;
Bootstrap – Layout, componentes e grid responsivo;

# Funcionalidades:

Layout completamente responsivo;
Navegação simples e intuitiva;
Seção de habilidades técnicas;
Exibição de projetos em desenvolvimento;
Seção “Sobre Mim”;
Botões de contato e redes sociais;
Organização e uso do grid do Bootstrap;

# Aprendizados:

Durante o desenvolvimento deste portfólio, pratiquei:
Estruturação de layouts com Bootstrap;
Responsividade avançada;
Organização de seções e hierarquia visual;
Melhor uso de variáveis, classes e seletores no CSS;
Manipulação de DOM com JavaScript;
Melhores práticas de HTML semântico;

# Melhorias Futuras:

Animações mais avançadas;
Integração com API para exibir projetos de forma dinâmica;
Otimizações de performance;
